# Resource Builder 👁‍🗨

인벤 KR 사이트에 SCSS를 적용시키기 위한 베이스 도구

## 사용 방법

- `$ npm install -g node-sass`
- `$ cd ./.resources/ && npm install`

### 기본 지원 명령어

#### SCSS 파일 watch 및 빌드

- `$ npm run css`
- `./package.json` 파일로 파일 트리 구성

#### Javascript Transpiler (업데이트 예정)

- `$ npm run js`
